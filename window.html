Add a robot by IP or url. For example, your robot may have the IP address 192.168.1.6 on your local network. Sometimes a robot's API is only available on a particular port. If this is the case you will need to specify the port number as well.

About

This app was designed as an expiriment into the concept of security for robots.

Social concept

Similar to how you can add your friends on a social network, you can add robots using this app. Once a request is sent to the robot, the robot itself can choose to allow the connection. If the user is not accepted the the cannot communicate with the robot. This also allows the robot to disconnect a user who man be misusing the control interface. The robot could also store a list of users who are allowed to connect or a list of blocked users.
